If you are creating a new translation, copy the `English` folder as a reference and give it the appropriate name.


`@ Font References.json` file is responsible for the primary values of the "Font" property of text elements.
- These fonts are also can be used in <font="Name"> tags in ui (And in default text in main menu of the program).
- As the "Font" value (In "Font Defaults" too) you also can use any ttf/otf font file path or name of font that installed on computer ("Times New Roman", "Arial", etc..).

"Margin" property is element offset from the following sides: [Left, Top, Right, Bottom].
- It can also have a single value that will be common to all sides (Just [5] as example).
